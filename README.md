AI Browser Agent
================

Короткий старт и описание проекта — автономный AI-агент, который управляет браузером через Playwright и принимает решения от LLM (через OpenAI API с function-calling). Проект демонстрирует архитектуру автономности: наблюдение за страницей, выбор действия и исполнение.

Установка

1. Скопируйте `.env.example` в `.env` и заполните `OPENAI_API_KEY` и при необходимости `MODEL`.
2. Установите зависимости:

```bash
npm install
npx playwright install
```

Запуск

- Обычный режим (ввод задачи в терминале):

```bash
npm start
```

- Демо (пример сложной задачи):

```bash
npm run demo
```

Как это работает (кратко)

- `src/browserController.js` — обёртка над Playwright: открывает браузер, извлекает «снимок» страницы (тексты и атрибуты видимых элементов) и выполняет действия (click/type/navigate/screenshot).
- `src/agent.js` — цикл автономного принятия решений: отправляет снимок в LLM (через OpenAI Chat Completions + functions), получает структурированное действие и просит `browserController` выполнить его. Повторяет до завершения или запроса дополнительной информации.
